#### 2. Recherche en fonction de multiple terms
---
<details>
<summary>TP</summary>


Nous savons que les tags sont stockés en tant que `keyword`.

```
GET /products/_search
{
  "query": {
    "terms": {
      "tags.keyword": [ "Soup", "Cake" ]
    }
  }
}
```

Similaire à la clause IN en SQL.
